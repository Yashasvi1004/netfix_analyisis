# Netflix Data Analytics Project

## Project Overview
This project performs exploratory data analysis (EDA) on a Netflix movie dataset using Python.  
The objective is to analyze genre distribution, audience engagement through vote counts and popularity, and trends in movie releases over time.

The project focuses on data cleaning, exploratory analysis, visualization, and clear communication of insights.

---

## Dataset Description
The dataset contains information related to Netflix movies, including release details, popularity metrics, voting information, and genres.

### Dataset Columns
- **Release_Date** – Release date of the movie  
- **Title** – Movie title  
- **Overview** – Short description of the movie  
- **Popularity** – Popularity score  
- **Vote_Count** – Number of votes received  
- **Vote_Average** – Average rating  
- **Original_Language** – Original language of the movie  
- **Genre** – Genre(s) associated with the movie  
- **Poster_Url** – URL to the movie poster  

---

## Tools and Libraries Used
- Python  
- Pandas – data manipulation and analysis  
- NumPy – numerical operations  
- Matplotlib – data visualization  
- Seaborn – statistical visualizations  

---

## Analysis Workflow
1. Data loading and initial exploration  
2. Data cleaning and preprocessing  
3. Exploratory data analysis (EDA)  
4. Visualization of key patterns  
5. Answering analytical questions based on the dataset  

---

## Key Questions Answered
- What is the most frequent genre of movies released on Netflix?  
- Which genre receives the highest number of votes?  
- Which movie has the highest popularity and what is its genre?  
- Which movies have the lowest popularity and what are their genres?  
- Which year has the highest number of movie releases?  

---

## Key Insights
- Drama is the most frequent and most voted genre.  
- Action-oriented movies dominate popularity rankings.  
- Movies with less common genre combinations show lower popularity.  
- Movie production peaked in the year 2022.  
- Visualizations strongly support numerical findings.

# 🐍 Python Data Analytics Projects

A growing collection of data analytics projects built with Python, exploring real-world datasets across various domains.

## 📁 Projects

### 🎬 Netflix
Exploratory data analysis on Netflix's content library — uncovering trends in content type, genres, release years, countries, and ratings.

### 🍽️ Zomato Project
Analysis of Zomato restaurant data to explore cuisine trends, city-wise distributions, pricing, ratings, and customer preferences.

### 🛒 E-Commerce
Data analysis of e-commerce transactions — examining sales patterns, customer behavior, product performance, and revenue trends.

## 🛠️ Tools & Libraries

| Tool | Purpose |
|------|---------|
| **Python** | Core programming language |
| **Pandas** | Data cleaning & manipulation |
| **NumPy** | Numerical computations |
| **Matplotlib** | Data visualization |
| **Seaborn** | Statistical plots & charts |
| **Jupyter Notebook** | Interactive analysis environment |
